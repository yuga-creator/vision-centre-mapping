# Changelog - Vision Centre Mapping Frontend

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-13

### Added
- **Interactive Map Screen**: Added Leaflet-like interactive maps with OSRM routing and custom polylines.
- **Nearby Lookup**: Automated search for nearby eye care centers sorting by ascending distance.
- **Glassmorphism Theme**: Added customized UI overlays with real-time blur and border decoration (`GlassCard`).
- **Passcode Protection**: Admin toggle in header using passcode verification to protect write actions.
- **Automatic Reverse Geocoding**: Coordinates generated dynamically during new center additions.
