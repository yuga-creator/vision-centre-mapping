# Changelog - Vision Centre Mapping Backend

All notable changes to the backend codebase and configuration files are listed below.

## [1.0.0] - 2026-06-13

### Added
- **CSV Bulk Uploader**: Created `uploader.html` with PapaParse for fast client-side CSV scanning and batch Firestore updates.
- **Firebase Configurations**: Added default security rules template `firestore.rules` and indexes blueprint `firestore.indexes.json`.
- **Backend Schema Docs**: Provided specifications for Firestore schema attributes and geo coordinates formatting.
- **CSV Sample Database**: Created `samples/centers_sample.csv` for bulk uploader testing.
- **Deployment Manuals**: Created step-by-step installation guides for new client onboarding.
